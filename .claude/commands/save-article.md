# 記事保存コマンド

指定されたURLの記事を取得し、Markdown形式で保存します。

## 引数

$ARGUMENTS: 保存する記事のURL

## 手順

### 1. 記事の取得

WebFetchツールを使用してURLから記事を取得する。

### 2. 言語判定と翻訳

- 記事が英語の場合：日本語に翻訳する
- 記事が日本語の場合：そのまま使用

### 3. ディレクトリ名の生成

記事タイトルから適切なディレクトリ名を生成する：
- 日本語タイトルはそのまま使用可
- 特殊文字は除去またはハイフンに置換
- 長すぎる場合は50文字程度に短縮

### 4. ファイル保存

以下の構成で保存：

```
repositories/articles/<ディレクトリ名>/
├── article.md      # 記事本文
└── source.txt      # 元URL（参照用）
```

### 5. article.md のフォーマット

```markdown
# <記事タイトル>

> 元記事: <URL>
> 取得日: <YYYY-MM-DD>

---

<記事本文>
```

## 出力

保存完了後、以下を報告：
- 保存先パス
- 記事タイトル
- 翻訳の有無
