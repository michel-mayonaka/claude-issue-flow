# 未実装項目一覧

`wiki/` ディレクトリ内のMarkdownファイルから未チェック項目（`- [ ]`）を抽出して一覧表示する。

## 手順

1. `wiki/` 内の全 `.md` ファイルを読み込む
2. `- [ ]` パターンを抽出
3. ファイル名・セクション名と共に一覧表示

## 出力フォーマット

```
## 未実装項目一覧

### Implementation-Status.md

#### Core機能 > Issue処理
- [ ] 後処理セクションのパース
- [ ] 後処理指定フォーマットのバリデーション

#### Core機能 > 実装
- [ ] サブエージェント呼び出し
...
```

## 実行

`wiki/` ディレクトリ内のすべての `.md` ファイルを読み込み、未チェック項目（`- [ ]`）を抽出して上記フォーマットで出力してください。

セクション見出し（`##`, `###`）を追跡して、各項目がどのセクションに属するかも表示すること。
